Assignment #3; Prolog and Constraint Satisfaction Problems (CSPs)
Total Marks: 10

Objective
Learn to model and solve a Constraint Satisfaction Problem (CSP) in Prolog using CLP(FD), and explore the effects of different search strategies on solution finding.

Background
A CSP involves assigning values from predefined domains to a set of variables such that all constraints are satisfied. CLP(FD) in Prolog allows you to:

Declare variable domains

Post constraints

Search for solutions using various strategies

Forward checking (constraint propagation) occurs automatically when constraints are posted, helping prune impossible assignments early.

Search for Solutions Using Different Strategies (3 Marks)

Use the following labeling strategies with labeling/2:

Default ([]): Assign variables left-to-right in ascending order.

First-Fail (ff): Choose the variable with the smallest remaining domain first.

Minimum (min): Choose the variable with the smallest possible value first.
